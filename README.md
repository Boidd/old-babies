A simple configurable client side mod to bring back the original look of baby mobs prior to the 26.1 baby mob overhaul.

This mod restores the original models of all baby mobs revamped in the 26.1 update 

There is a configuration screen accessible using [Mod Menu](https://modrinth.com/mod/modmenu) that allows enabling/disabling the classic models of individual mobs

The models of baby rabbits are unchanged as their default model was updated

The mod is compatible with resource packs that change mob textures

Releases are available on modrinth: https://modrinth.com/project/old-babies
